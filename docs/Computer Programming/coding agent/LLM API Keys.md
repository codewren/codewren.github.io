# Free LLM API keys

## Gemini API key

1. Step-by-Step Guide to Getting Your Key
Visit Google AI Studio: Go to [aistudio.google.com](aistudio.google.com).

Sign In: Use your standard Google/Gmail account.

Create API Key:

On the left sidebar, click on "Get API key".

Click the button "Create API key in new project".

Copy and Save: Copy the string immediately. For security, never check this key into GitHub; use an .env file instead
