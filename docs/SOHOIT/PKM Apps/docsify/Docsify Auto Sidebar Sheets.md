# Docsify-auto-sidebar Quick Sheet

The **docsify-auto-sidebar** tool is a Node.js utility that automatically generates a `_sidebar.md` file by scanning your project's local directory structure.

---

### Top 5 Most Used Commands

| Command | Action |
| --- | --- |
| `npx docsify-auto-sidebar` | Generates the `_sidebar.md` based on your current folder structure. |
| `docsify-auto-sidebar -d ./docs` | Specifies a **custom directory** (`./docs`) to scan for Markdown files. |
| `docsify-auto-sidebar -t "Title"` | Sets a custom **sidebar title** at the top of the generated list. |
| `docsify-auto-sidebar -l 2` | Limits the scanning depth to a specific **directory level**. |
| `docsify-auto-sidebar --help` | Lists all available flags and configuration options for the tool. |
